# Iris Flower Classification 🌸

## 📌 Overview
This project is a Machine Learning model that classifies iris flowers into different species based on their petal and sepal measurements.

The project uses the famous Iris Dataset and applies machine learning algorithms to predict flower species accurately.

---

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset Information
The dataset contains 150 samples of iris flowers with the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

1. Import Dataset
